# Visualisation des performances sportives (Python)

Projet académique : analyse d’un fichier CSV (temps, distance) et calcul de la vitesse et de l’accélération, avec visualisation.

## Fichiers
- `devoir5.py` : programme Python
- `performance_athlete.csv` : données d’entrée
- `rapport_devoir5.pdf` : rapport du projet

## Fonctionnalités
- Lecture et validation du fichier CSV
- Calcul de la vitesse et de l’accélération (dérivées discrètes)
- Visualisation graphique des résultats
